Tailwind CSS 学习和探索

发表于2022-12-26|更新于2022-12-27
|阅读量:

Tailwind CSS 是一个功能类优先的 CSS 框架，它集成了诸如 flex, pt-4, text-center 和rotate-90这样的的类，它们能直接在脚本标记语言中组合起来，构建出任何设计。

安装

使用 Tailwind CLI 工具 安装：

```
npm install -D tailwindcss
npx tailwindcss init
``
